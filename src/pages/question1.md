# 더하기
## 문제 설명
주어진 리스트 내에 홀수를 찾아 합을 반환하는 solution 함수를 완성해주세요.

---

## 제한 사항

- 1 ≤ 리스트 안의 수 ≤ 100
- 숫자 외에 다른 값은 주어지지 않습니다.
- 각각의 값은 1차원 리스트로 주어집니다.
- 리스트 안에 값이 없는 경우는 0으로 출력해야 합니다.

---

## 입출력 예

|   입력    | 출력 |
| --------- | ------ |
| [1, 2, 3, 4, 5] | 9 |
| [3, 5, 7, 1] | 16 |

---

## 입출력 설명
[1, 2, 3, 4, 5] 중에서 홀수는 [1, 3, 5]입니다. 이 숫자를 모두 더하면 9가 됩니다.
